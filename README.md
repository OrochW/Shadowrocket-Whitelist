## 📱 自动更新（iOS快捷指令）

一键配置每日自动更新规则：  
[🔥 获取快捷指令](https://www.icloud.com/shortcuts/7323c4ebc30c4e7095184d766177a118)

**功能说明**：
- 每天自动检查规则更新
- 更新成功发送通知
- 支持手动立即更新

**配置步骤**：
1. 点击上方链接安装快捷指令
2. 进入「快捷指令」App → 自动化 → 创建个人自动化
3. 选择「特定时间」设置每日更新时间（推荐凌晨3点）
4. 添加操作 → 选择「运行快捷指令」→ 选中本指令
5. 关闭「运行前询问」

## 🛠️ 使用方法

1. **初次使用**：
   - 下载规则文件 或 扫码导入
   - 打开 Shadowrocket → 配置 → 启用本规则

2. **日常维护**：
   - iOS用户：通过快捷指令自动更新
   - 其他用户：重新下载规则文件
